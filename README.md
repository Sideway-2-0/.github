# 🌀 Sideway 2.0

_„Tam, kde se cesty rozdělují, realita ustupuje neznámu.“_

**Sideway 2.0** je zatím **tajemstvím** zahaleným mlhou. Na první pohled obyčejný svět, ve skutečnosti ale plný skrytých možností a příběhů, které čekají, až je někdo objeví. Tento projekt je připraven převrátit zažitá očekávání a poskytnout ti možnost prozkoumat nové obzory.

> Vstup jen na vlastní nebezpečí. **Sideway 2.0** odměňuje ty, kteří se odváží vykročit mimo vyšlapané cesty a objevovat, co bylo dosud skryto. Buď trpělivý, přípravy probíhají.
